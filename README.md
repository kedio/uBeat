team5
=====
First time installation:
----
```
npm install -g bower
```
After every pull:
----
```
bower install
npm install
```
If we use the local UBeat server, we have to create a new user each time. Go to [http://localhost:3000/signup](http://localhost:3000/signup) to do that.
To run the web server:
----
```
node server.js
```
